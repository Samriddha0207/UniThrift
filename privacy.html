How We Protect It

  • All data is stored on Supabase with Row Level Security (RLS) — you can only access your own data
  • Verification documents (College ID, PAN card) are stored in private buckets — not publicly accessible
  • Authentication uses industry-standard JWT tokens with short expiry and automatic refresh
  • All traffic is encrypted via HTTPS/TLS
  • We never sell your data to third parties

AI Processing

When you submit verification documents, they are sent to Google Gemini AI for automated review. Gemini checks whether the document appears to be a genuine College ID or PAN card. The image is processed in real-time and is not stored by Google for training purposes under their API terms.

The AI result (verified/rejected + confidence score) is stored in your profile. The actual document image stays in our private Supabase storage bucket and is only reviewed by UniThrift administrators.

Third Parties

  • Supabase — database, auth, storage (GDPR compliant)
  • Google Gemini API — document verification AI (see Google's privacy policy)
  • Google Fonts / FontAwesome — UI assets (CDN, no personal data)

We do not use advertising networks, analytics trackers, or sell data to any marketing platforms.
